Three arms per side — control (always take the systematic pick), overlay treatment (VETO → cash for that slot), cascade (VETO → walk to next-best, always invested). Kill switch: 12 months / 30 picks, drop if scores don't predict returns OR treatment doesn't beat control. (HANDOFF, memory `llm_overlay_experiment`)
