# Addendum (same day): L1 fix shipped + pre-2026-05-30 deep audit

## L1 upper-price filter — IMPLEMENTED, and it caught a live contaminant

Added `universe.MAX_PRICE_USD = 5000` + `HIGH_PRICE_ALLOWLIST` (NVR, SEB,
BKNG, BRK-A, AZO, MELI). A price > cap on `as_of` is rejected unless
allowlisted (a bare threshold can't separate BKGM's $10k ghost from NVR's
real ~$9.9k ATH, so the allowlist is required). `max_price=0` disables it.

**This was NOT just preventative — it caught real backtest contamination.**
Building the allowlist empirically (every ticker that ever closed >$4k)
revealed BKGM was ranking **#1 by momentum across all of 2025_H1** at
~$10,000 (real history $14–$11k). i.e. the momentum sleeves were *buying a
data ghost and booking phantom returns* in backtests.

### Frozen regression — RE-BASELINED (legitimately, corruption removal)
2025_H1 only (2023_Q4 unaffected — BKGM's jump wasn't in that lookback):
| | old (contaminated) | new (clean) |
|---|---|---|
| momentum_v1 2025_H1 | +2.1865% / 154 | **+2.3022% / 153** |
| momentum_v2 2025_H1 | +12.1738% / 90 | **+12.4171% / 93** |
All 4 tests now pass at **d=0.0000pp** on the clean baseline.

### Held-out (2024-01→2026-05) contamination — measured, BOUNDED & SMALL
| strategy | contaminated | clean | BKGM inflation |
|---|---|---|---|
| **mom_roa_6535** (deployed) | +36.47% CAGR | **+36.47%** | **0.00pp** |
| mom_v2 | +28.82% CAGR | **+27.99%** | −0.83pp |

**The deployed champion (mom_roa_6535) is completely unaffected** — its
dual-signal rank_fn requires BOTH momentum and a sane ROA (|ROA|≤1), and a
ghost has no valid ROA, so it was never picked. mom_v2 (momentum-only) was
inflated by a modest 0.83pp; corrected figure **+27.99% CAGR held-out**.

### Live paper trade — CLEAN
BKGM was **never held** by any live sleeve. On 2026-05-01 (paper inception)
it ranked #2742 (flat at $10k → no momentum), so it was never picked. The
contamination existed only in backtests whose lookback spanned the jump.
